# ❌⭕ Morpion – Jeu en Python

## 📌 Objectif
Créer un jeu de morpion en terminal (joueur vs IA ou 2 joueurs)

## 🧰 Stack
- Python 3

## 🚀 Lancer le jeu
```bash
python3 morpion.py
```

## ✅ Fonctionnalités
- Affichage en grille
- IA (facile, moyenne, difficile)
- Option 2 joueurs

## 📚 Commandes
Pas de dépendances externes, pur Python.
